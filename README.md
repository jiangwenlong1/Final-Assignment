# Expense Tracker

## Project Description

Expense Tracker is a simple Python application that helps users record and analyze daily expenses. The program allows users to enter expense categories and amounts, calculates total spending, identifies the highest expense category, and generates visual charts.

## Project Purpose

The purpose of this project is to help users manage personal finances and understand their spending habits through data visualization.

## Features

* Record expenses by category
* Calculate total expenses
* Identify the highest expense category
* Generate a pie chart for expense distribution
* Generate a bar chart for expense comparison

## Tools and Libraries

* Python 3
* Matplotlib

## Installation

Install the required library:

pip install -r requirements.txt

## How to Run

python expense_tracker.py
